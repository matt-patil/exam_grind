# Exam Grind

Exam Grind is a task-based alarm application specifically designed for students preparing for rigorous exams like JEE and NEET. The app ensures you wake up and get your brain active by requiring you to complete missions (tasks) like solving simple MCQ's before the alarm can be dismissed.

## Key Features

- **Mission-Based Alarms**: Forget simple snooze buttons. You must complete a challenge to turn off the alarm.
- **Multiple Mission Types**:
  - **Math Challenge**: Solve mathematical problems to wake up your brain.
  - **Typing Challenge**: Type out specific phrases or quotes.
  - **Shake Challenge**: Physically shake your phone a certain number of times.
  - **MCQ Challenge**: Answer multiple-choice questions. Contains specially curated datasets for JEE preparation (Physics, Chemistry, Math for 11th grade).
- **Customizable Alarms**: Set alarms for specific days, choose different sounds, and adjust the volume.
- **Motivational Quotes**: Wake up to motivational quotes tailored to keep your preparation on track.
- **Sleek Dark UI**: A modern, distraction-free interface built with Flutter.
